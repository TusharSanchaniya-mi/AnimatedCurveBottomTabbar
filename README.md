## Example

To run the example project, clone the repo, and run `pod install` from the Example directory first.

``
pod 'AnimatedBottomTabbar', :git => 'https://github.com/TusharSanchaniya-mi/AnimatedBottomTabbar.git', :tag => '1.0.0'
``
